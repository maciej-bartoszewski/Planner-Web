# Planner-Web
## Uruchamianie projektu

Aby uruchomić projekt należy mieć zainstalowane: `Node.js` oraz `npm`, następnie należy wykonać następujące komendy:
1. ```git clone https://github.com/maciej-bartoszewski/Planner-Web```
2. ```cd Planner-Web```
3. ```npm install```
4. ```npm run dev```
